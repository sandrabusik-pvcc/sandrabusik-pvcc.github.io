TERM

DEFINITION
Machine language Sequence of instructions for modern computers encoded as numeric digits.
debugging locating and correcting coding errors.
assembler program that converted mnemonic expressions into machine language instructions
assembly language A mnemonic system for representing programs; machine language program
translator high-level primitives are translated into machine-language programs.
compiler Compiled several machine instructions into short sequences to simulate the activity reuested by a single high-level primitive.
interpreter Executed the instructions as they were translated instead of recordin the translated version for future use.
cross_platform software Software that is independent of an operating system's design as well as the machine's hardware design and is therefore executable throughout a network.
natural language Such as English, German, and latin and evolved over time without formal grammatical analysis.
formal language Such as programming languages and are precisely defined by grammars.
machine independence Third-generation languages allowed for this as much as possible.
high-level programming language Provide more abstrations for the programmer and amke it easier for people to read and write a program.
object-oriented programming (OOP) A programming paradigm that is the most prominent associated with the programming process called object-orientated programming (OOP). Following this paradigm, a software system is viewed as a collection of units, called objects, each of which is capable of performing the actions that are immediately related to itself as well as requesting actions of other objects. Together, these objects interact to solve the problem at hand.
scripting language Used to perform administrative tasks rather than to develop complex programs.
data types: The manner in which the data item is encoded and the operations that can be performed on hat data.
  • integer: Numeric data consisting of whole numbers, probably stored using two's complement notation. Operations that can be done on integer data include the traditional arithmetic operations and comparisons of relative size, such as determining whether one value is greater than another.
  • float: Numeric data that might contain values other than whole numbers.
  • string: A sequence of characters.
primitive data type predefined by the language and is named by a reserved word.
data structure The conceptual shape or arrangement of data. For ex, text is normally viewed as a lon string of characters.
Python collection data structures
  • list: Used to store multiple items in a single variable but they are ordered, changeable, and allow duplicate values. They are written with []. They are indexed with the first item [0] and second item [1]
  • tuple: Used to store multiple items in a single variable but they are ordered, unchangeable, and allow duplicates. They are written with (), round brackets. They are indexed with the first item [0] and second item [1]
  • set: Used to store multiple items in a single variable but they are unordered and unchangeable and do not allow duplicates. They are written with {}, curly brackets.
  • dictionary: Used to store data values in key:value pairs but they are ordered, changeable, and do not allow duplicates. Written with curly brackets and have keys and values.
function A set of instructions for performing a task that can be used as an abstract tool by other program units.
local variable A variable declared within a function, meaning that it can be referenced only within that function.
global variable Variables whose scopes are not restricted to a particular part of a program.
source program The program in its original form when it is being translated.
object program The translated form of a program as apposed to its original form.
