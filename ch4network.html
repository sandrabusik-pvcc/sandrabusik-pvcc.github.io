DEFINITION
Network Classification A computer network is classified as being either a PAN; personal area network which is between a wireless mouse and its PC. or any of the following three:
  • LAN:local area network; a collection of computers in a single building or building complex.
  • MAN:metopolitan area network;collection of computers spanning a local community.
  • WAN:wide area network; Links machines ove neighboring cities or on opposite sides of the world
Network models Pattern in which machines are connected or the topology of the network.
  • bus network:machines are all connected to a common communication line called a bus.
  • star network: one machine serves as a central focal point to which all the others are connected.
Protocols Rules by which activies are conducted
  • CSMA/CD Ethernet: Carrier Sense, ultiple Access with Collision Detection. The right to transmit messages protocol. This protocol dictates that each message be broadcast to all the machines on the bus. Each machine monitors all the messages but keeps only those addressed to itself. To transmit a message, a machine waits until the bus is silent. IT begins transmitting while continuing to monitor the bus. If another machine also egins transmitting, both machines detect the clash and pause for a brief random period of time before trying to transmit again.
  • CSMA/CA wifi: Carrier Sense, Multiple Access with Collision Avoidance. Wireless networks try to avoid collisions rather than trying to detect them. Referred to as WiFi. When collisions do occur, messages must e retransmitted.
Combining networks Sometimes it is necessary to connect existing networks to form an extended communication system.
  • repeater: a device that passes signals back and forth between the two original buses, usually with some amplification, without considering the meaning of the signals.
  • bride:It connects two buses, but only forwards the messaeacross the connection only when that messae is destined for a computer on the other side by lookingat the destination address the accompanies each message.
  • switch:A bridge with mutiple connections. Each message is relayed only into the appropriate spoke, thus minimizing the traffic in each spoke.
  • router:devices that handle the connections between networks that form an internet.
  • gateway: point at which one network is linked to an internet serving as a passaeway between the network and the outside world.
Client/server model An interprocess communication convention which defines the basic roles played by the processor as either a client, which makes requests of other processes, or a server, which satisfies the reuests made by clients.
  • P2P:peer-to-peer:another interprocess communication convention model. Involves processes that provide service to and receive service from each other.
  • distributed system:Consist of software units that execute as processes on different computers.
  • cloud computing: huge pools of shared computers on the network can e allocated for use by clients as needed. The latest trend in distributed systems.
Internet Addressing The internet is a collection of connected networks. The networks are constructed and maintained by organizations called Internet Sperice Providers(ISPs). These system of networks can e classified in a heirarchy.
  • Tier 1:Top of heirarchy. Relatively few of them. High speed, high capacity, international WANs. Thought of as the backbone of the Internet.
  • Tier 2:Connecting to the tier-1 ISPs are the tier-2 ISPs. Tend to be more regional in scope and less potent in their capabilities.
  • Tier 3: Provides access to the core which is Tier 1 and 2. An access. An independent Internet; an intranet
IP address Internet needs an internet-wide addressing system that assigns a unique identifying address to each computer in the system.
  • ICANN: Non-profit corporation established to coordinate the Internet;s operation. It awards blocks of consecutive IP addresses to ISPs.
  • DNS:Domain name system. Internet-wide directory system which converts a computer name humans use to its computer IP.
  • TLD:Top-level domain:A name's suffix following the period in an internet address, such as UVA.edu. It is used to reflect the domain's classification, which in this case is educational.
  • Country-code TLDs: Countries' all over the world suffix.
Internet protocols What transfers something over the internet.
  • FTP:FIle Transfer Protocol. An application for listin and copying files across the network.
  • HTTP:Hper Text Transfer Protocol. Most sophisticated.
  • SMTP:Simple Mail Transfer Protocol.
World wide web Origins in the work of Time Berners-Lee, combined internet technology with linked documents; hypertext.
  • URL:Uniform Resource Locator. In order to locate and retrieve documents on the Web, each document is iven a unique address called a Uniform Resource Locator (URL).
  • Client-side activities:Activites done on a webpage, if handles by the client such as the browser.
  • Server-side activities: Activities done on a webpage, if handled by the server such as a webserver.
Internet four-layer software Software for controlling communication over the internet has four layers consisting of the application layer, the transport layer, the network layer, and the link layer.
  • application layer:Consists of those sotware units such as clients and servers that use Internet communication to carry out their tasks. May use the services of the name servers within the Inernet to translate mnemonic addresses used by humans into Internet-Compatible IP addresses
  • transport layer:Sends and recieves messages over the Internet
  • network layer:Decide in which direction a placket should be sentat each step.
  • link layer:Layer at the routerin charge of receiving and transmittin the packets.
Internet is: Links a variety of computer networks so that they could function as a connected system that would not be disrupted by local disasters.
  • packet-swtching network:Transport layer.
  • fault tolerant:Most or all of the network remains connected even when very important tier-1 routers fail.
  • redundant: Many possible networks of routers in Tier 1 and 2 that provides the Internet's communication infrastructure so most or all of the network remains connected even when very important tier-1 routers fail.
Cyersecurity The study of protecting computers, networks, and data from unauthoried access and vandalism, forms of cybercrime.
  • malware:malicious software
  • virus:Inserts itself into programs that already reside in the machine.
  • worm:Autonomous program that transfers itself though a network, taking up residence in computers and forwarding compies of itself to other computers.
  • Trojan horse:A program that enters a computer system disuised as a desirable program, such as a game or useful utility package that is willingly imported by the victim.
  • spyware:sniffing software. Software that collects information about activites at the computer on which it resides and reports that information back to the instigator of the attack.
  • phishing:Obtaining information explicitly by simple asking for it.
  • DOS attack:Process of overloading a computer with messages.
Protection Ways to protect a computer from threats.
  • firewall:Filter traffic passing through a point in the network with a program called a firewall.
  • spam filter:A variant of a firewall for a specific purpose blocking unwanted email.
  • proxy server:A software unit that acts as an intermediary etween client and a server with the oal of shielding the client from adverse actions of the server.
  • antivirus software:Used to detect and remove the presence of known viruses and other infections. It is a broadd term for detecting and removing various kinds of infections.
  • cryptoraphy:the sutdy of sending and receiving messages securely in the presence of adversaries.
  • public key encryption:the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
